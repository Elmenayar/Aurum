# Security Policy

## Supported Versions

We currently support the latest production version of First Agent only.

| Version | Supported |
| ------- | --------- |
| Latest  | ✅        |
| Older versions | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in First Agent, please report it by email:

security@aurum-eg.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Screenshots if available
- Possible impact

We will review valid reports and respond as soon as possible.

Please do not publicly disclose the vulnerability until it has been fixed.
